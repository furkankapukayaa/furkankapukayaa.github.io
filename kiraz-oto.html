Kiraz Oto - Desktop Automation System

Developed a comprehensive desktop application tailored for Kiraz Oto to streamline their daily operations. The system includes robust modules for inventory management, employee tracking, sales records, customer relationship management, and service logging.

This solution empowers the business to maintain precise control over their workflow and data, significantly reducing manual paperwork and improving overall service delivery efficiency.


Technologies Used

  • C#
  • .NET
  • SQL Server
  • Windows Forms
