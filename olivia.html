Olivia Hair Mansion - Salon Automation System

Built a specialized automation software for Olivia Hair Mansion to organize and accelerate their daily salon operations. The platform offers intuitive features for appointment scheduling, customer tracking, staff management, and strict financial control.

By digitalizing their services and sales tracking, the salon team can now seamlessly manage incoming payments, monitor staff performance, and focus more on delivering premium experiences to their clients.


Technologies Used

  • .NET
  • SQL Server
  • Desktop Dev
